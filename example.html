Generating HTML

Processing the slides using theme from Discount, a Markdown implementation in C:

    theme -c -superscript -f slides.theme example.md

See also Makefile to automate this further.

Organisation

  • Each H2 header introduces a new slide
  • The result is a single HTML file
  • Scrolling snaps to slides
